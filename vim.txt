:q => quit
:wq => save & quit
:q! => quit w/o saving

jklh => move
gg => move top
G => move down
zz => center cursor to page
w => move through words right
b => move through words left 
$ => end line
A => go to end of line + insert mode
t + where => go where you want///; to next it

u => undo
ctrl+r => redo
. => do again

dd => delete + copy
y => copy
p => paste
v => visual mode/select
shift+v => per line 'v'

dw => delete word
cw => change word
x => delete character
r => replace character

o => newline 
O => newline up
> => tab
< => reverse tab

/ => search word
     n => cycle through founds

comment code block => ctrl+v -> shift+i -> // -> esc
